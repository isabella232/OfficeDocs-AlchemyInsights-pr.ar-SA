# <a name="microsoft-store-support"></a>دعم Microsoft Store

نحن نستمع ونريد المساعدة. إذا كنت بحاجة إلى مساعدة في العثور على معلومات حول شراء منتج أو حالة الطلب أو مشاكل الفوترة أو العروض الترويجية الخاصة أو معلومات تحديد موقع المنتج أو دعم Microsoft Store، فشاهد الاتصال بدعم [Microsoft Store](https://support.microsoft.com/account-billing/contact-microsoft-store-support-4f615f2a-6bbd-fd69-6695-ae213d63eef0).

للتعرف على الأجهزة التي تم إصدارها حديثا أو المنتجات القادمة من Microsoft، راجع [Microsoft Store](https://www.microsoft.com/?ql=1).

شكرا لكونك أحد عملاء Microsoft.
