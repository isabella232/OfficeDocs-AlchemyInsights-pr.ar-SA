# <a name="remove-a-hub-site"></a>إزالة موقع مركز

إلغاء تسجيل موقع مركز في مركز إدارة SharePoint الجديد:
  
1. في مركز إدارة SharePoint الجديد، ضمن **المواقع،** انقر فوق **المواقع النشطة**.

2. حدد الموقع، وانقر فوق **موقع Hub**، ثم انقر فوق إلغاء **التسجيل كموا موقع hub**.

3. انقر فوق **موافق**.

[معرفة المزيد حول مواقع الموزع](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f)
