# <a name="sharepoint-time-zone-settings"></a>SharePoint المنطقة الزمنية

بختيار مالك الموقع أو مسؤول مجموعة المواقع SharePoint المناطق الزمنية (الإعدادات الإقليمية) لكل مستخدمي الموقع. يمكن للمستخدمين الفرديين اختيار استخدام الإعدادات الافتراضية للموقع أو تحديد الإعدادات الشخصية الخاصة بهم. 

- **تغيير المنطقة الزمنية على مستوى الموقع**: راجع تغيير المنطقة [الزمنية أو الإعدادات الإقليمية الأخرى](https://support.microsoft.com/office/change-regional-settings-for-a-site-e9e189c7-16e3-45d3-a090-770be6e83c1a). 
- **تغيير المنطقة الزمنية على مستوى المستخدم**: راجع [تغيير اللغة الشخصية وإعدادات المنطقة](https://support.microsoft.com/office/change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).
