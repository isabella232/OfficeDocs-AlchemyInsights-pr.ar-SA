# <a name="install-project-professional"></a>تثبيت Project Professional

راجع [تثبيت Project](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) للتعرف على كيفية تثبيت إصدار الاشتراك من Project Online، والإصدار بدون اشتراك من Project 2019 أو 2016 أو 2013. 

بالنسبة إلى Project 2010 أو 2007، اتبع الخطوات في [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) أو [Office 2007.](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657) 

يتم بيع Project كتطبيق منفرد، وبالتالي إذا لم يكن لديك بعد، فجرب [Project أو شراؤه.](https://www.microsoft.com/evalcenter/evaluate-project)
